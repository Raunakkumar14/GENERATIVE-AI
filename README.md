# GenAI Fundamentals Toolkit

A Python-based Generative AI toolkit built to explore and understand core concepts of Large Language Models (LLMs), embeddings, and conversational AI through hands-on implementation.

This project combines multiple small experiments into a single unified repository to demonstrate how different components of modern GenAI systems work.

---

## 🚀 Features

This project includes three main components:

### 1. 🧠 Movie Information Extractor
- Extracts structured movie details from unstructured text
- Uses LLM prompting with schema validation
- Ensures consistent output using Pydantic models

### 2. 🔍 Text Embeddings Generator
- Generates vector embeddings using Google Gemini
- Converts text into numerical representations
- Useful for semantic search and similarity tasks

### 3. 💬 Conversational Chatbot
- Stateful chatbot using LLMs
- Maintains conversation history
- Supports different personality modes:
  - Angry mode
  - Funny mode
  - Sad mode

---

## 🛠️ Tech Stack

- Python
- LangChain
- Mistral AI (Chat Model)
- Google Gemini (Chat + Embeddings)
- Pydantic (Data validation)
- dotenv (Environment management)
